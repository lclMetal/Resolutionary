Reference

Resolutionary is a system for scaling your games made with Game Editor.

resChangeActorType(const char *actorName, int type)

actorName - name of the actor
type - the type to be set for the actor

  • RES_ACTOR_DYNAMIC
  • RES_ACTOR_STATIC
  • RES_ACTOR_NOT_DRAWABLE

Example call:

resChangeActorType("Event Actor", RES_ACTOR_DYNAMIC);

This function sets an actor's type that specifies how Resolutionary should handle the actor.

resChangeActorZLayer(const char *actorName, int zLayer)

actorName - name of the actor
zLayer - the number of the z layer to assign to the actor

This function sets an actor's z layer. Actors are drawn in order by their z layer, from lowest to highest. So, actors with a higher z layer will be drawn on top of the actors with a lower z layer. The number of an actor's z layer can be positive or negative
