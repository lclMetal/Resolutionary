resActorDetector

resActorDetector is used for detecting actors that are currently on screen. This actor should be exactly the same size as the view actor is in the original resolution of the game, so that the area that is monitored is the same as the original view size.

Type

  • Filled Region actor

Back to top

resBackground

resBackground is used as a background that hides the actual game under the Resolutionary screen.

Type

  • Normal actor

Back to top

resBorders

Two clones of this actor are used as the "black bars" for filling the leftover space on the screen if the aspect ratios of the original resolution and the new resolution don't match.

Type

  • Normal actor

Back to top

resDynamicLayerCanvas

resDynamicLayerCanvas is used for drawing the dynamic actors and is updated every frame.

Type

  • Canvas actor

Back to top

resMouse

resMouse is an actor the size of one pixel and is used for detecting which actors are found at the position where the mouse should be in relation to the original resolution.

Type

  • Normal actor

Back to top

resMouseClickDetector

resMouseClickDetector is used for capturing mouse clicks. When clicked, the actor sends an Activation Event to every actor currently colliding with resMouse.

Type

  • Filled Region actor

Back to top

resStaticLayerCanvas

resStaticLayerCanvas is used for drawing the static actors and is updated only when requested or when a change occurs in the list of static actors, i.e. an actor is either added to the list or removed from it.

Type

  • Canvas actor
